# Emacs 覚書

## キーバインド

| Key     | Description  |
| ------- | ------------ |
| `C-M-s` | 正規表現検索 |
| `M-s w` | 単語検索     |
| `C-q`   | 特殊文字入力 |

## 便利コマンド

| Name                                               | Description           |
| -------------------------------------------------- | --------------------- |
| `align-regexp`                                     | 列揃えの整形          |
| `M-: (byte-recompile-directory "~/.emacs.d/" 0 t)` | 全Elispの再コンパイル |
